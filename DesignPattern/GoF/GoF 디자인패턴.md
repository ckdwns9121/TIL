# GoF 디자인 패턴의 분류

논문 **"Using Pattern Languages for Object-Oriented Programs"**를 통해 제안되었다.

## 생성 패턴(Creational Pattern)

- 추상 팩토리
- 빌더
- 팩토리 메서드
- 프로토타입
- 싱글톤

## 구조 패턴(Structual Pattern)

- 어댑터
- 브릿지
- 컴포지트
- 데코레이터
- 파사드
- 플라이웨이트
- 프록시

## 행위 패턴(Behavioral Pattern)

- 책임 연쇄
- 커맨드
- 인터프리터
- 이터레이터
- 중재자
- 메멘토
- 옵저버
- 상태
- 전략
- 템플릿메서드
- 방문자
